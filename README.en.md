# Open Computer Science Courses from Universities Worldwide

[中文版 README](https://github.com/0voice/Awesome-OpenCS)

A curated collection of open computer science courses from top universities worldwide (such as MIT, Stanford, Berkeley) and renowned online education platforms (such as Coursera, edX). Covering core areas of computer science including programming fundamentals, data structures and algorithms, computer systems, artificial intelligence and machine learning, databases and big data, networking and security, software engineering and development, computer graphics, quantum computing, theory of computation, and blockchain. Each category provides a course table listing course names, sources, languages, difficulty levels, and valid links, supplemented with introductions and explanations to guide beginners and advanced learners in selecting suitable courses and navigating subtopics. All links have been verified (as of October 21, 2025) to ensure resource availability, making it suitable for self-learners, students, and developers to explore computer science theory and practice in depth.

## Contribution Guidelines

Welcome to contribute courses or course materials to **Awesome-OpenCS**! Please follow these steps:

1. Fork this repository.
2. Add courses in `README.md` (maintaining the table format) or add course materials (PDFs, code, etc.) in the corresponding subfolders under `resources/`.
3. Submit a Pull Request, explaining the added content.

Please ensure that courses are free, links are valid, and course material filenames are clear (e.g., `MIT_6-0001_Lecture1.pdf`).

## Resource Notes

- **Course Sources**: International (MIT OCW, Coursera, edX).
- **Course Material Storage**: Lecture notes, assignments, code, etc., are stored in [resources/](resources/), categorized by topic. They need to be downloaded from the resource page and uploaded to the corresponding subfolder.
