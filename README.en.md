## Contribution Guidelines

Welcome to contribute courses or materials to **Awesome-OpenCS**! Follow these steps:

1. Fork this repository.
2. Add courses to `README.md` (maintain table format) or materials (PDFs, code) to `resources/` subfolders.
3. Submit a Pull Request describing your additions.

Ensure courses are free, links work, and filenames are clear (e.g., `MIT_6-0001_Lecture1.pdf`).
